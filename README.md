# 2026 문화데이터 및 신기술 활용 공모전 요강

## 1. 공모전 소개
**기술과 문화의 융합, 더 나은 미래를 설계할 여러분의 창의적인 아이디어를 기다립니다.**

본 공모전은 데이터 활용 역량을 높이고 AI 기반 서비스 개발을 촉진함으로써, 우리 문화 생태계의 경쟁력을 강화하고 새로운 활력을 불어넣고자 마련되었습니다. 기술과 문화의 시너지를 통해 혁신적인 가치를 창출할 여러분의 열정적인 참여를 진심으로 기다립니다.

## 2. 참여 대상
* **대한민국 국민 누구나** 참여 가능
* **개인 또는 단체** (5인 이내의 팀 / 기관 및 기업 참가 가능)
* ⚠️ *주의: 동일 작품으로 여러 부문에 중복 응모는 불가합니다.*

## 4. 접수 방법
* **양식 다운로드:** 공모전 공식 홈페이지 ([culture.go.kr/digicon](https://culture.go.kr/digicon))에서 참가 신청 서류 양식을 다운로드
* **제출 방식:** 작성 후 홈페이지를 통해 온라인 제출
* **제출 서류:**
  1. 참가신청서
  2. 기획서 (또는 분석보고서)
  3. 참가 서약서
  4. 개인정보 수집 및 이용에 관한 동의서
* **파일 제출 규격:** * 모든 신청 서류는 양식에 맞춰 작성 후 **PDF 파일로 변환**합니다.
  * 변환된 PDF 파일들을 **1개의 압축파일(.zip)**로 묶어서 제출합니다.
  * **파일명 예시:** `분야_부문_팀명.zip` (예: `우수사례_디지털혁신_이윤희.zip`)
* ⚠️ *주의: 작년도 양식과 차이가 있으므로, 반드시 **2026년도 최신 양식**을 다운로드하여 작성해 주시기 바랍니다.*
